# weather_forecast
天气查询小程序
其中b.png是可执行程序的背景图片
forecast.py是带图形化的天气查询程序（需要python3环境）
ic.ico是可执行文件的图标
ss.qrc是通过qt导入的图片文件打包可执行程序的时候，转换的一个包含在python中的一个模块
tianqi.ui是通过qt设计的图形化界面
weather_search.py是不带图形化的天气查询程序
